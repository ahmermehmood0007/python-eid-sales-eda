# 🛍️ Eid Sales Data Analysis using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on an Eid Sales dataset using Python.

The objective is to analyze customer purchasing behavior, product categories, sales distribution, and key business insights through data visualization.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

Eid Sales Dataset

---

## Analysis Performed

- Data Loading
- Data Cleaning
- Missing Value Analysis
- Duplicate Check
- Data Exploration
- Customer Demographics
- Product Category Analysis
- State-wise Sales Analysis
- Age Group Analysis
- Gender-wise Purchasing
- Top Selling Products
- Data Visualization

---

## Key Insights

- Identified the highest selling product categories.
- Compared purchasing behavior by gender.
- Analyzed sales across different states.
- Explored customer age distribution.
- Visualized overall sales trends.
